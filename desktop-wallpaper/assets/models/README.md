# 3D Model Assets

Place production character assets here:

- `xiao-qi.glb` for the female companion
- `xiao-qi-native-v4.glb` / `xiao-qi-native-v4-mobile.glb` for the integrated
  female body, head, neck, rig, and vetted motion library
- `qi-an.glb` for the male companion
- `<character>-walk.glb` / `<character>-run.glb` for rigging defaults
- `<character>-wave.glb`, `<character>-nod.glb`, `<character>-heart.glb`, and `<character>-voice.glb` for interactive full-body actions
- `<character>-idle.glb` is kept as an optional generated idle clip, but the runtime currently uses a calmer sampled standing pose
- optional `.fbx` files for DCC tools and animation cleanup

The current modeling pipeline is documented in `modeling/MODELING_PIPELINE.md`.
